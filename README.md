## Installation

Install my-project with yarn

```bash
  yarn install
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/Capstone3-BlockConnect/Frontend.git
```

Go to the project directory

```bash
  cd blockConnect
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
